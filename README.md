## Report Structure

Each report follows this professional structure:

```
Executive Summary          ← Non-technical summary for leadership
Vulnerability Overview     ← CVE details, affected versions
Technical Analysis         ← Mechanism, attack vector, impact
Threat Actor Context       ← ATT&CK TTPs, actor profiling
Detection Opportunities    ← IOCs, SIEM queries
Prioritized Patch Plan     ← Immediate → short-term → long-term actions
Risk Rating Summary        ← CVSS, exploitability, business impact
References                 ← NVD, vendor advisories, standards
```
