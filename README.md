## About

This repository contains structured vulnerability analysis reports demonstrating professional threat intelligence analyst capabilities. Each report is written to serve two audiences simultaneously: **executive leadership** (CISO/CEO) requiring risk context and business impact, and **technical teams** requiring actionable detection and remediation guidance.

Reports follow industry-standard formats and frameworks:

- **MITRE ATT&CK** — TTP mapping from initial access through exfiltration
- **CVSS v3.1** — Severity scoring with full vector string breakdown
- **TLP (Traffic Light Protocol)** — Appropriate classification for each product
- **NIST SP 800-40** — Patch management prioritization methodology
